## Features
* **Offline-First Vault:** Fully functional without an internet connection.
* **Smart Dashboard:** Integrated Pomodoro timer, dynamic task list, and calendar event tracker.
* **Rich Text Workspace:** Custom note editor with dynamic tags, pinning, and trash management.
* **Memory Optimized:** Custom garbage collection and process-killing logic to keep RAM usage strictly below 200MB.

## Tech Stack
* **Framework:** Electron.js
* **Frontend:** HTML5, JavaScript (ES6+), Tailwind CSS
* **Database:** Local JSON File System

## How to Run Locally
**1.** Clone the repository
**2.** Install dependencies: `npm install`
**3.** Start the application: `npm start`
**4.** Build the executable: `npx electron-packager . Smart-Notes --platform=win32 --arch=x64 --overwrite`
